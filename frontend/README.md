# ArXiv Daily web frontend

React, TypeScript, Tailwind CSS, and Vite frontend for browsing the `papers.json` generated by `scripts/fetch_papers.py`.

## Local development

```bash
npm ci
npm run dev
```

`frontend/public/papers.json` is sample data used only for local development. The Pages workflow builds the app and then replaces the built copy with the freshly validated `data/papers.json`.

## Verification

```bash
npm run lint
npm run build
```

The Vite base path is derived from `GITHUB_REPOSITORY` during GitHub Actions builds, so forks and renamed repositories do not need source changes. Set `VITE_BASE_PATH` explicitly only for a custom hosting layout.
